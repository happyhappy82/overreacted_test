## 챗GPT 항공권 프롬프트 예시와 활용 팁

### 프롬프트 예시 1: 기본적인 항공권 검색

가장 기본적으로 필요한 정보를 얻기 위한 프롬프트입니다. 출발지와 목적지 여행 날짜를 명확히 입력하는 것이 핵심입니다.

```plain text

[출발 날짜]에 출발하여 [도착 날짜]에 돌아오는 항공편 정보를 찾아줘.
```

예산을 정해두었거나 특정 경유 횟수를 선호하는 경우 조건을 추가하여 검색할 수 있습니다.

```plain text

[예산 범위] 내에서
최대 [경유 횟수] 이하인 항공편을 찾아줘.
```

### 프롬프트 예시 3: 특정 항공사 및 시간대 선호 검색

선호하는 항공사가 있거나 특정 시간대에 출발 도착하고 싶을 때 유용한 프롬프트입니다.

```plain text

가능하다면 [오전/오후/저녁] 출발편을 우선으로 찾아줘.
```

### **프롬프트 예시 4: 유연한 날짜 검색 (최저가)**

정확한 날짜가 정해지지 않았고 최저가 항공권을 찾고 싶을 때 활용할 수 있습니다.

```plain text

```

