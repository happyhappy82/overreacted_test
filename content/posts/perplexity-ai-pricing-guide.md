# 퍼플렉시티 AI 구독료, 가격 총정리

## 요약

[퍼플렉시티 AI](https://www.perplexity.ai/)는 검색 기능의 강점을 살린 AI 툴입니다.

무료 버전은 기본적인 정보를 찾는 데 유용하지만 심화된 검색 기능을 활용하려면 유료 플랜이 유리합니다.

학생 할인이나 연간 결제 방식을 통해 비용 부담을 줄일 수 있는 선택지도 마련되어 있습니다.

### 핵심 체크리스트

무료 버전의 Pro 검색 횟수는 하루 5회로 제한됩니다.

- 개인 Pro 구독료는 월 20달러이며 연간 결제 시 200달러로 비용이 낮아집니다.
- 대학생과 교직원은 SheerID 인증을 거쳐 월 4.99달러에 이용이 가능합니다.
- 파워 유저를 위한 Max 플랜은 월 200달러의 높은 비용이 발생합니다.
- 기업용 요금제는 팀 협업과 데이터 보안에 중점을 둔 기능을 포함합니다.
- API 서비스는 구독료와 별개로 사용량에 기반한 과금 방식이 적용됩니다.

## 퍼플렉시티 AI 개인용 및 교육용 요금제 분석

개인 사용자가 선택할 수 있는 플랜은 크게 세 가지 유형으로 구분할 수 있습니다. 무료 버전인 Free 플랜은 기본적인 검색을 제공하지만 고급 모델을 사용하는 검색 횟수가 상당히 적습니다.

반면 Pro 플랜은 월 20달러의 비용으로 다양한 AI 모델 선택과 파일 업로드 기능을 제공하여 업무 효율을 높여줍니다. 특히 대학생이나 교직원의 경우 인증을 통해 월 4.99달러라는 합리적인 가격으로 Pro 기능을 동일하게 누릴 수 있습니다.

고성능 검색이 일상화된 사용자라면 무료보다는 유료 구독이 실질적인 도움을 줄 것으로 생각합니다.

## 기업용 플랜과 API 사용 가이드

조직 단위로 서비스를 이용하고자 한다면 Enterprise 플랜이 적절한 대안이 될 수 있습니다.

Enterprise Pro는 인당 월 40달러의 비용으로 데이터 보안이 중요한 환경에서 팀원들과 협업할 수 있도록 도와주기 때문입니다.

더 높은 수준의 리서치 기능이 필요한 조직은 Enterprise Max를 통해 제한을 줄인 환경을 구축할 수 있습니다.

단, 개발자를 위한 API는 구독 서비스와는 다른 별도의 과금 체계를 가지고 있어 주의가 필요합니다. API는 요청 횟수와 토큰 사용량에 따라 비용이 발생하며 Search API의 경우 1,000회 요청당 5달러 수준입니다. 사용 중인 Pro 구독이 있다면 월 5달러의 API 크레딧이 기본 제공되므로 이를 먼저 활용해 보는 것이 좋습니다.

**요약포인트**: 조직용 플랜은 보안과 협업에 집중하며 API는 사용량만큼 비용을 지불하는 구조입니다.

## 자주 묻는 질문(Q&A)

**Q. 무료 버전의 Pro 검색 5회는 언제 초기화되나요?**

A. 일반적으로 24시간을 기준으로 초기화가 진행됩니다. 하루에 정해진 횟수를 모두 사용하면 다음 날 다시 이용할 수 있는 방식입니다.

**Q. 교육용 할인은 모든 학생에게 적용되나요?**

A. SheerID 인증 시스템을 통해 대학급 이상의 학생이나 교직원으로 확인되어야 합니다. 초중고 학생의 경우 인증 여부에 따라 제한이 있을 수 있습니다.

**Q. Pro 플랜 구독 중에도 API 사용료를 따로 내야 하나요?**

A. Pro 구독자는 매달 5달러의 API 크레딧을 받게 됩니다. 해당 크레딧을 초과하는 사용량에 대해서는 별도의 요금이 청구됩니다.

**Q. 기업용 플랜은 데이터 학습을 정말 하지 않나요?**

A. Enterprise 요금제는 사용자의 검색 데이터나 업로드된 파일을 모델 학습에 사용하지 않는 것을 원칙으로 합니다. 이는 기업의 기밀 정보를 보호하기 위한 조치로 이해할 수 있습니다.

**Q. 결제 후 구독을 해지하면 바로 기능이 차단되나요?**

A. 해지를 하더라도 이미 결제한 기간까지는 Pro 기능을 계속 사용할 수 있습니다. 다음 결제 예정일에 맞춰 자동으로 갱신이 중단되는 구조입니다.
