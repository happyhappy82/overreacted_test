[노트북LM](https://notebooklm.google.com/) 동영상 개요 기능은 텍스트 소스를 기반으로 설명 영상을 제작해 주는 서비스입니다. 이 기능을 활용하면 복잡한 문서 내용을 시각적인 영상 자료로 편리하게 변환할 수 있습니다. PC 버전에서 스타일과 형식을 자유롭게 설정할 수 있으며 모바일에서는 브라우저 접속을 통해 이용 가능합니다.



맞춤형 프롬프트를 사용하면 청중의 수준이나 보고 목적에 맞는 최적화된 결과물을 얻을 수 있습니다.



## **핵심포인트**

- 소스 문서를 추가한 후 Studio 패널에서 생성 버튼 클릭하기
- 설명 또는 요약 중 원하는 영상 형식 선택하기
- 수채화/ 레트로풍 등 10가지 이상의 시각적 스타일 적용해 보기
- Steering Prompt를 사용하여 특정 섹션 강조나 대상 지정하기
- 생성된 영상은 링크로 공유하거나 MP4 파일로 직접 다운로드 가능
- 모바일 앱 대신 크롬이나 사파리 브라우저를 통해 웹 버전으로 접속하기


방대한 양의 문서를 읽고 핵심을 정리하는 과정은 많은 시간과 노력이 들어가는 작업입니다.

최근에는 텍스트를 단순히 요약하는 수준을 넘어 시각적인 영상 형태로 변환해 주는 툴이 주목받고 있습니다. 특히 유튜브에 동영상 개요를 통해 만든 영상이 올라오는 것들을 종종 볼 수 있는데요.



## **PC 웹 환경에서의 동영상 개요 사용법**

먼저 PC 웹 브라우저에서 노트북LM 사이트에 접속하여 새 노트북을 생성하거나 기존 파일을 불러옵니다.

![Image](/notion-images/노트북lm-동영상-개요-사용법pc모바일-image-1.webp)

동영상개요 버튼을 클릭합니다.

해당 메뉴를 선택하고 생성 버튼을 누르면 문서 내용을 바탕으로 영상 제작이 시작됩니다.

![Image](/notion-images/노트북lm-동영상-개요-사용법pc모바일-image-2.webp)

참고로 동영상 오버뷰는 맞춤설정이 가능합니다.

설명 동영상 / 요약 중에서 형식을 선택할 수 있고시각적 스타일선택도 가능합니다.

![Image](/notion-images/노트북lm-동영상-개요-사용법pc모바일-image-3.webp)

스타일에는 기본/화이트보드/귀여움/일본 애니메이션/수채화/레트로풍/전통/종이공예 등이 있습니다. 이중에서 원하는 컨셉을 선택해서 진행하시면 됩니다.

![Image](/notion-images/노트북lm-동영상-개요-사용법pc모바일-image-4.webp)

동영상 개요를 보시면 다큐멘터리 영상을 깔끔한 유튜브 애니메이션 영상콘텐츠 형식으로 바꿔주는 모습을 알 수 있습니다. 심지어 음성도 실제 배우가 녹음한것처럼 자연스러워 퀄리티 높은 영상을 볼 수 있습니다.

## **동영상 개요 설정 및 비교 기준**

| 항목 | 내용 | 비고 |
| --- | --- | --- |
| 형식 | Explainer(상세) / Brief(요약) | 정보 밀도·영상 길이 조절 |
| Visual Style | Anime / Watercolor / Classic 등(약 10종) | 영상 전반 시각 분위기 결정 |
| Steering Prompt | 사용자 지시문 입력 칸 | 맞춤형 콘텐츠 제작 핵심 도구 |
| Output Language | 한국어 및 다국어 선택 가능 | 구글 계정 설정 언어 기반 |
| 공유 방식 | 링크 공유 / MP4 다운로드 | 기업용 계정은 외부 공유 제한 가능 |

## **모바일에서 노트북LM 동영상개요 만드는법**

더 나은 결과물을 얻기 위해서는 설명 칸에 구체적인 지시 사항을 입력하는 것이 좋습니다.

예를 들어 입문자를 대상으로 한 교육영상이라면 쉬운 비유를 섞어달라는 지시를 추가할 수 있습니다.

업무 보고용 영상을 제작할 때는 결론과 수치를 먼저 제시하도록 설정하는 방식이 효과적이라고 생각합니다.

참고로 현재 노트북LM 모바일 앱에서는 동영상 개요 기능을 직접적으로 지원하지 않고 있습니다.

따라서 스마트폰이나 태블릿에서 이 기능을 쓰려면 브라우저를 통해 웹 버전으로 접속해야 합니다.

## Q&A

Q. 영상 생성 시간은 보통 어느 정도 걸리나요?

A. 소스 문서의 분량과 복잡도에 따라 다르지만 보통 수 분 이내에 완료되는 편입니다. 백그라운드에서 작업이 진행되므로 기다리는 동안 다른 작업을 계속하실 수 있습니다.

Q. 한국어 음성으로 영상을 만드는 것이 가능한가요?

A. 네 설정 메뉴의 Output Language에서 한국어를 선택하면 한국어 내레이션이 적용된 영상이 생성됩니다. 기본적으로 구글 계정에 설정된 기본 언어를 인식하여 작동합니다.

Q. 기업용 워크스페이스 계정에서 공유가 안 되는 이유는 무엇인가요?

A. 기업용 또는 교육용 계정은 보안상의 이유로 외부 링크 공유 기능이 비활성화되어 있을 수 있습니다. 이럴 때는 영상을 직접 다운로드하여 파일 형태로 공유하는 방법을 권장합니다.

Q. 비주얼 스타일은 어떤 것들이 있으며 추천하는 스타일이 있나요?

A. 수채화/레트로풍/전통 등 10가지 이상의 스타일이 존재합니다. 공식적인 보고용으로는 화이트보드가 적합하며 가벼운 교육용으로는 기본 스타일을 추천합니다.

Q. 특정 페이지 내용만 영상으로 만들 수 있나요?

A. Steering Prompt에 “문서의 3장 내용만 중점적으로 다뤄달라”는 지시를 넣으면 가능합니다. 툴이 해당 지시를 인식하여 특정 섹션에 집중한 결과물을 만들어줍니다.
