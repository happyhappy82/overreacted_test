## **핵심 요약**

[노트북LM](https://notebooklm.google.com/)은 사용자가 제공한 소스를 분석하여 퀴즈와 플래시카드를 자동으로 만들어주는 툴입니다.

PC 웹과 모바일 앱 환경 모두에서 Studio 패널을 통해 접근할 수 있으며 이동 중에도 학습이 가능합니다.

난이도와 문항 수를 사용자가 직접 설정할 수 있어 수준별 맞춤 학습 환경을 제공합니다.

## **핵심 체크리스트**

- 학습에 사용할 소스 파일을 노트북에 먼저 업로드하기
- 화면 우측 또는 하단의 Studio(스튜디오) 패널 접속하기
- 퀴즈(Quizzes) 항목을 선택하여 문제 세트 생성하기
- 문항 수(fewer/standard/more)와 난이도(easy/medium/hard) 조절하기
- 모바일 앱에서는 연필 아이콘을 눌러 세부 설정 변경하기
- 정답 확인 시 Explain 버튼을 활용하여 상세 설명 듣기
## **노트북LM 퀴즈 기능은 어떤건가요?**

많은 양의 정보를 학습할 때 단순히 읽는 것보다 문제를 풀며 점검하는 방식이 기억 유지에 도움이 됩니다.

노트북LM은 이러한 학습 과정을 지원하기 위해 소스 기반의 퀴즈 생성 서비스를 제공하고 있습니다.

복잡한 설정 없이도 몇 번의 클릭만으로 텍스트를 문제화할 수 있어 학습 효율을 높일 수 있다고 생각합니다.

지금부터 PC와 모바일 기기에서 각각 이 기능을 사용하는 구체적인 절차를 알아보겠습니다.

## **PC에서 노트북LM 퀴즈 사용법**

먼저 노트북LM에 들어갑니다. 원하시는 자료를 추가해주세요.



![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-1.webp)

자료 업로드가 완료되면 화면 오른쪽 레이아웃에 위치한 Studio(스튜디오) 패널로 이동합니다.

여기서 퀴즈를 클릭합니다.



![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-2.webp)

참고로 질문수를 더적게/표준/더많이와 난이도를 쉬움/중간/어려움으로 설정할 수 있습니다.


![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-3.webp)



퀴즈가 만들어지면 문제를 풀 수 있습니다. 노트북LM 퀴즈 기능의 장점이 오답을 선택할때 어떤게 오답이고 정답인지 구분을 해주고 각각 이유를 설명해주는 것이 유용합니다.


## **모바일 앱에서 노트북LM 퀴즈 사용법**

안드로이드나 아이폰 사용자는 NotebookLM 앱을 실행한 후 해당 노트북의 Studio 탭으로 들어갑니다.



![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-4.webp)



모바일 버전에서도 동일하게 Quizzes 메뉴를 통해 문제 세트를 만들거나 이전에 만든 퀴즈를 열 수 있습니다.


![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-5.webp)



![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-6.webp)

모바일 화면에서 연필 모양의 아이콘을 누르면 퀴즈에 사용할 언어와 소스 범위를 세밀하게 조정할 수 있습니다.


![Image](/notion-images/노트북lm-퀴즈-사용법pc모바일-image-7.webp)



마찬가지로 모바일에서도 문제를 풀면 각 이유를 알려줘서 효율적인 학습이 가능합니다.
특정 챕터만 문제로 풀고 싶다면 소스 선택 메뉴에서 필요한 파일만 체크하여 문제를 생성하면 됩니다.



### 모바일 퀴즈 실행 시 체크리스트


- 앱 스토어를 통해 최신 버전으로 업데이트가 되었는지 확인하십시오.
- 퀴즈 생성 전 연필 아이콘에서 대상 소스가 올바르게 선택되었는지 점검하십시오.
- 언어 설정이 의도한 대로 되어 있는지 확인하여 오답률을 줄이십시오.
- 학습 결과 리포트를 통해 취약한 부분을 다시 점검하는 시간을 가지십시오.


요약: 모바일 앱은 연필 아이콘을 통한 직관적인 설정 변경으로 장소에 상관없이 학습하기 편리합니다.

## **Q&A**

Q. 퀴즈 생성 메뉴가 보이지 않는데 어떻게 해야 하나요?

A. 이 기능은 현재 순차적으로 적용 중일 수 있으므로 앱이나 브라우저를 최신 버전으로 업데이트해야 합니다. 또한 소스를 추가한 직후에는 시스템이 데이터를 분석하는 인덱싱 시간이 필요할 수 있으므로 잠시 기다려 보시기 바랍니다.

Q. 퀴즈의 난이도가 너무 낮게 느껴지면 조정이 가능한가요?

A. 설정 메뉴에서 난이도를 hard로 변경하여 더 복잡한 문제를 생성할 수 있습니다. 모바일의 경우 연필 아이콘을 눌러 직접 프롬프트 지시어를 추가하여 문제의 수준을 높여달라고 요청하는 방법도 있습니다.

Q. 한 번 생성한 퀴즈는 저장되어 나중에도 볼 수 있나요?

A. 네 Studio 패널에서 생성된 모든 퀴즈는 아티팩트 형태로 저장되어 언제든지 다시 열어볼 수 있습니다. 따라서 시험 기간이나 복습이 필요한 시점에 저장된 퀴즈를 다시 호출하여 학습에 활용하는 것이 가능합니다.
